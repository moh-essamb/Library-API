## **API Collection Overview** This API allows you to manage **Books**, **Households**, **Users**, and **Wishlists** with basic CRUD operations (Create, Read, Update, Delete). Each operation requires a valid `g-token` for authentication. ## 🌐 Base URL ``` http://localhost:3000 ``` ## **Books API** - **POST**: Create a book with title, author, and year. - **GET**: Retrieve a book by its `id`. - **PUT**: Update book details. - **DELETE**: Delete a book by `id`. --- ## **Households API** - **POST**: Create a household with a name. - **GET**: Retrieve a household by its `id`. - **PUT**: Update the household's name. - **DELETE**: Delete a household by `id`. --- ## **Users API** - **POST**: Create a user with `firstName`, `lastName`, and `email`. - **GET**: Retrieve a user by `id`. - **PUT**: Update user details. - **DELETE**: Delete a user by `id`. --- ## **Wishlists API** - **POST**: Create a wishlist with a name and list of books. - **GET**: Retrieve a wishlist by `id`. - **PUT**: Update wishlist details (name or books). - **DELETE**: Delete a wishlist by `id`. --- ### **Common Response Codes** - **200 OK**: Success. - **201 CREATED**: Resource created. - **204 NO CONTENT**: Deletion successful (no content returned). - **404 NOT FOUND**: Resource not found. - **500 INTERNAL SERVER ERROR**: Server error.
